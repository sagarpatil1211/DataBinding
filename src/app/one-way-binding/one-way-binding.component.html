I. One way data binding

One-way data binding is a one-way interaction between component and its template. If you perform any changes in your component, then it will reflect the HTML elements.

1. String interpolation

In general, String interpolation is the process of formatting or manipulating strings.

In Angular, Interpolation is used to display data from component ( TS file ) to view ( HTML DOM ).

It is denoted by the expression of {{ symbol }} and also known as mustache syntax.

Ex.1 Show data = {{name}}

Ex.2 Change data using setInterval : {{no}}

2. Property binding

Property binding is used to bind the data from property of a component to DOM elements. It is denoted by [ ].

Ex.1 use value property of input tag to show name string variable

Ex.2 use disabled property of button to change state of button (setInterval) see console log

3. Event binding

Event binding is used to bind the data from DOM HTML element to component

Events are actions like mouse click, double click, hover or any keyboard and mouse actions.

If a user interacts with an application and performs some actions, then event will be raised.

It is denoted by either parenthesis ( )
